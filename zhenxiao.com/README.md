### BTC密码学原理

主要密码学中的两个功能：哈希和签名

密码学中用到的哈希函数称为：cryptographic hash function，其有两个重要的性质：

collision resistance  x!=y 使得 H(x) = H(y) ，这个很难避免，输入空间近乎无限，输出空间是有限的，根据鸽笼原理,必然会产生碰撞。这里要说的是，当我们给定一个x，没有什么比较好的方法让我们找到一个y来让 H(x) = H(y)  brute-force
